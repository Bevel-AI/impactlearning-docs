# Audio Generation

Impact Learning can generate professional voice narration for your content. Add audio to create more engaging and accessible learning experiences.

## Supported Providers

### ElevenLabs

ElevenLabs provides industry-leading text-to-speech with natural-sounding voices.

| Model | Description |
|-------|-------------|
| **ElevenLabs v3** | Latest model with enhanced quality and natural prosody |

**Features:**
- Multiple voice options
- Language support for global audiences
- Voice settings (stability, similarity, style)
- Natural pauses and emphasis

### OpenAI TTS

OpenAI offers text-to-speech with instruction support for tone control.

| Model | Description |
|-------|-------------|
| **GPT-4o Mini TTS** | GPT-4o powered speech synthesis |

**Features:**
- Instruction-based tone control
- Consistent voice quality
- Fast generation

## How It Works

1. **Select text for narration** — Identify content that needs audio
2. **AI generates speech** — The selected model converts text to natural speech
3. **Audio is embedded** — Automatically added to your learning content
4. **Captions included** — Accessibility captions are generated automatically

## Voice Customization

### ElevenLabs Settings

| Setting | Description | Range |
|---------|-------------|-------|
| **Stability** | How consistent the voice sounds | 0-1 |
| **Similarity Boost** | Match to the selected voice | 0-1 |
| **Style** | Expressiveness of delivery | 0-1 |
| **Speaker Boost** | Enhance speaker clarity | On/Off |

### OpenAI Instructions

With OpenAI TTS, you can provide instructions to control delivery:

```
Speak in a calm, encouraging tone suitable 
for a meditation and wellness course.
```

```
Use an energetic, motivating voice for this 
fitness training introduction.
```

## Use Cases

### Narrated Lessons
Add voiceover to explain complex concepts while learners view visual content.

### Accessibility
Provide audio alternatives for learners who prefer listening or have visual impairments.

### Language Learning
Create pronunciation guides and listening exercises with natural speech.

### Scenario Dialogue
Generate character voices for role-play simulations and branching scenarios.

<Callout type="info">
Audio narration significantly improves engagement and accessibility. Consider adding narration to key sections of your learning content.
</Callout>
